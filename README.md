# Div Follows Mouse Cursor 🖱️✨

An interactive web application that creates a playful effect where a div element smoothly follows your mouse cursor across the screen. Perfect for adding a fun, dynamic element to your website! 🎯💫

- Responsive design
- Easy to integrate into any web project
